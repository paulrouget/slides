1compilation

  • install prerequisites
  • hg clone http://hg.mozilla.org/mozilla-central/
  • edit mozconfig
  • make -f client.mk
  • wait
  • execute: firefox -P dev -no-remote

7find/file a bug

  • What do you want to do? Should it be an addon?
  • Open a bug (even if duplicate)
  • learn how to use bugzilla (but not too much)
  • Don't know where to start?
    • is.gd/goodfirstbug
    • is.gd/goodfirstbug_mentor
    • ask us!

8communicate!!!!!

  • Tell us you want to work on something
  • to get guided, to avoid frustration
  • know who's working on what
  • IRC (#fx-team)
  • mailing list (dev.apps.firefox)
  • bugzilla (Firefox component)
  • twitter (follow me!)
  • Find a mentor!

8mercurial

  • learn mercurial (and mq)
  • to create a patch
  • to organize your patches
  • ... or git (but Mercurial is easier)
is.gd/hack_firefox_mercurial

9patch / review

  • You have some code, you have a bug
  • You need to get the code in Firefox
  • You need positive reviews (r+)
  • upload your patch
  • find a reviewer (irc is your friend)
  • ask for review (r?)
  • Not sure? Ask for feedback (f?)
is.gd/hack_firefox_review
